## What is Dizilo

Dizilo builds custom AI agents, intelligent workflow automations, and e-commerce solutions for businesses that are tired of doing everything manually.

We replace repetitive, time-consuming work with technology that runs 24/7 — so business owners focus on growth, not admin.

---

## Services

| Service | What it does |
|---|---|
| AI Agents | Intelligent systems that think, act, and respond automatically |
| Workflow Automation | End-to-end business processes that run without human input |
| E-commerce Solutions | Shopify & WooCommerce stores built to convert and scale |
| Lead Generation Agents | Capture, qualify, and follow up leads around the clock |
| Customer Support Bots | Handle 80% of queries automatically |
| Invoice & Payment Automation | Auto-generate, send, chase, and reconcile invoices |
| Document Generation | Auto-create contracts, proposals, and reports |
| CRM Automation | Auto-log calls, update pipelines, send follow-ups |

## Tech Stack

- **Framework** — Next.js 16 (App Router)
- **UI** — React 19 + Tailwind CSS 4
- **Language** — TypeScript 5
- **Hosting** — Vercel
- **Automation** — n8n (self-hosted)
- **AI** — OpenAI GPT-4o, gpt-image-1
- **Media** — Cloudinary
- **Analytics** — Google Search Console

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

---

## Environment Variables

Create a `.env.local` file in the `agenix/` directory. See `.env.example` for required variables.

> All API keys and automation scripts are excluded from this repository via `.gitignore` and are never committed.
